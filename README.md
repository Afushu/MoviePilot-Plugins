# MoviePilot-Plugins

MoviePilot插件集合，包含各种实用插件。

## 插件列表

### MediaManager
- **名称**：MediaManager
- **版本**：1.0.0
- **作者**：Afushu
- **描述**：媒体资源管理和播放插件，支持多网盘集成、STRM文件生成和视频播放
- **标签**：媒体管理、网盘、STRM、视频播放

## 如何使用

1. 在MoviePilot管理界面中，进入"设置" → "插件市场"
2. 在"插件仓库地址"中添加：`https://github.com/Afushu/MoviePilot-Plugins`
3. 点击"保存"并刷新插件列表
4. 找到所需插件并点击"安装"
